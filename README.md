# 🌟 Rakshita Gupta | Product Manager

Welcome to the repository for my personal product portfolio! This website is designed to be a highly interactive, aesthetic, and tactile showcase of my journey as a Product Manager and Co-founder. 

---

## ✨ Project Overview

This portfolio steps away from traditional, rigid layouts and instead embraces a warm, scrap-book-style aesthetic. It features skeuomorphic elements (like a physical pocket/wallet holding cards), custom typography, and smooth scroll animations to create a memorable user experience.

### 🎨 Key Features
* **Interactive Hero Section:** A custom-built "wallet" interface where hovering over the pocket smoothly fans out polaroids, ID cards, and maps.
* **Tactile UI Elements:** Buttons and badges designed with 3D press effects to mimic physical objects.
* **Smooth Scroll Reveals:** Content beautifully glides and fades into the viewport as the user scrolls, powered by the Intersection Observer API.
* **Fully Responsive Grid:** A mobile-friendly layout ensuring a seamless experience across desktop, tablet, and mobile screens.
* **Warm, Grainy Aesthetic:** Custom background grid, radial glows, and carefully chosen earthy color palettes (Sand, Olive, Orange, Forest Green).

---

## 🛠️ Tech Stack

This project is built using pure, vanilla web technologies to ensure lightweight performance and zero dependencies.

* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom variables, Flexbox, CSS Grid, complex hover states, cubic-bezier transitions, and animations.
* **Vanilla JavaScript:** Intersection Observers for scroll reveals and smooth anchor scrolling.
* **Fonts:** *DM Serif Display*, *Bricolage Grotesque*, and *Roboto* via Google Fonts.
